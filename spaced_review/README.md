# Spaced Review

Weekly review quizzes generated by LLM.

## How to generate
Tell Claude: "今週のreviewを作って" → Claude reads `questions/index.md` + each `analysis.md` and generates a quiz file here.

## Quiz format
Each weekly file contains:
- Problem title + link
- Quiz: identify Layer 3 pattern / Layer 2 technique / Layer 1 data structure
- Problems marked `needs_review: true` are prioritized
- Link included for problems you want to code immediately

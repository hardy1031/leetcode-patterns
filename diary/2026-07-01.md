## Theme: Dummy Node — Parallel list traversal with carry

Solved 2 (Add Two Numbers). Same structure as 21 (Merge Two Sorted Lists) — dummy node, traverse both lists simultaneously, build a new list. The only new element is carry handling: when digit sum >= 10, carry 1 to the next iteration. After the loop, if carry remains, append a final node.

### Key insight
The dummy node pattern generalizes to any problem where you build a new linked list node-by-node — not just deletion, but construction too.

## Theme: BST In-Order Traversal

BST in-order traversal (left → root → right) produces nodes in ascending sorted order. This makes it the go-to technique for any BST problem that involves sorted access to elements.

### Key insight
**BST in-order = sorted ascending.** The BST property `left < root < right` directly translates to the traversal order. Iterative implementation uses an explicit stack: push all left children, pop and process, then move to right child.

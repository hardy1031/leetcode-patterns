# 15. 3Sum

Given an integer array `nums`, return all triplets `[nums[i], nums[j], nums[k]]` such that `i != j != k` and `nums[i] + nums[j] + nums[k] == 0`. Solution set must not contain duplicate triplets.

**Example 1:** nums = [-1,0,1,2,-1,-4] → [[-1,-1,2],[-1,0,1]]
**Example 2:** nums = [0,1,1] → []
**Example 3:** nums = [0,0,0] → [[0,0,0]]

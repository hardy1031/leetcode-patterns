# 167. Two Sum II - Input Array Is Sorted

Given a **sorted** (non-decreasing) integer array `numbers`, return 1-indexed `[index1, index2]` such that `numbers[index1-1] + numbers[index2-1] == target`. Use O(1) extra space.

**Example 1:** numbers = [1,2,3,4], target = 3 → [1,2]

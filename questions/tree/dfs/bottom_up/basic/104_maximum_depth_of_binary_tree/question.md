# 104. Maximum Depth of Binary Tree

**Difficulty:** Easy
**Pattern:** bfs_dfs
**LeetCode:** https://leetcode.com/problems/maximum-depth-of-binary-tree/
**NeetCode:** https://neetcode.io/problems/depth-of-binary-tree

## Problem

Given the root of a binary tree, return its depth. Depth is the number of nodes along the longest path from root to the farthest leaf.

## Examples

```
Input: root = [1,2,3,null,null,4]
Output: 3

Input: root = []
Output: 0
```

## Constraints

- `0 <= number of nodes <= 100`
- `-100 <= Node.val <= 100`

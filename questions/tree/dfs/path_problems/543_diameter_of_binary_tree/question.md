# 543. Diameter of Binary Tree

**Difficulty:** Easy
**Pattern:** tree/dfs/path_problems
**LeetCode:** https://leetcode.com/problems/diameter-of-binary-tree/
**NeetCode:** https://neetcode.io/problems/binary-tree-diameter

## Problem

Given the root of a binary tree, return the length of the diameter of the tree. The diameter is the length of the longest path between any two nodes (the path does not need to pass through the root).

## Examples

```
Input: root = [1,2,3,4,5]
Output: 3  (path: 4→2→1→3 or 5→2→1→3)

Input: root = [1,2]
Output: 1
```

## Constraints

- `1 <= number of nodes <= 10^4`
- `-100 <= Node.val <= 100`

# 238. Product of Array Except Self

Return an array `answer` where `answer[i]` is the product of all elements except `nums[i]`. O(n) time, no division.

**Example 1:** nums = [1,2,3,4] → [24,12,8,6]
**Example 2:** nums = [-1,1,0,-3,3] → [0,0,9,0,0]

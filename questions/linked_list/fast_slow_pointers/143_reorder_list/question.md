# 143. Reorder List

**Difficulty:** Medium
**Pattern:** linked_list/fast_slow_pointers

## Problem

You are given the head of a singly linked-list. Reorder the nodes to be in the following order:

[0, n-1, 1, n-2, 2, n-3, ...]

You may not modify the values in the list's nodes, but instead you must reorder the nodes themselves.

## Examples

**Example 1:**
```
Input:  head = [2,4,6,8]
Output: [2,8,4,6]
```

**Example 2:**
```
Input:  head = [2,4,6,8,10]
Output: [2,10,4,8,6]
```

## Constraints

- `1 <= Length of the list <= 1000`
- `1 <= Node.val <= 1000`

# 19. Remove Nth Node From End of List

**Difficulty:** Medium
**Pattern:** linked_list/two_pointers

## Problem

Given the head of a linked list and an integer n, remove the nth node from the end of the list and return its head.

## Examples

**Example 1:**
```
Input:  head = [1,2,3,4], n = 2
Output: [1,2,4]
```

**Example 2:**
```
Input:  head = [5], n = 1
Output: []
```

**Example 3:**
```
Input:  head = [1,2], n = 2
Output: [2]
```

## Constraints

- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`

# Questions Index

LLM向け: このファイルで全問題を把握し、各問題の詳細は `{pattern}/{number}_{name}/` 以下のファイルを参照。

## How to use (for LLM)
- Weekly review生成時: このindexを読む → `needs_review: true` の問題を優先 → 各`analysis.md`を読んでquizを生成
- 新問題追加時: このindexに行を追加する

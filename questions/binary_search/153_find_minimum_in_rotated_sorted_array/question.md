# 153. Find Minimum in Rotated Sorted Array

Given a sorted array rotated 1–n times, find the minimum element. Must run in O(log n).

**Example 1:** nums = [3,4,5,1,2] → 1
**Example 2:** nums = [4,5,6,7,0,1,2] → 0

# 918. Maximum Sum Circular Subarray

Given a circular integer array `nums`, return the maximum possible sum of a non-empty subarray.

**Example 1:** nums = [1,-2,3,-2] → 3
**Example 2:** nums = [5,-3,5] → 10
**Example 3:** nums = [-3,-2,-3] → -2

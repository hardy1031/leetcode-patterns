# 1071. Greatest Common Divisor of Strings

For two strings s and t, "t divides s" means s = t + t + ... + t. Given str1 and str2, return the largest string x that divides both.

**Example 1:** str1 = "ABCABC", str2 = "ABC" → "ABC"
**Example 2:** str1 = "ABABAB", str2 = "ABAB" → "AB"
**Example 3:** str1 = "LEET", str2 = "CODE" → ""

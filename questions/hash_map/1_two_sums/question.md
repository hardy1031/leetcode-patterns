# 1. Two Sum

Given an array of integers `nums` and an integer `target`, return the indices of two numbers that add up to `target`.

**Example 1:** nums = [2,7,11,15], target = 9 → [0,1]
**Example 2:** nums = [3,2,4], target = 6 → [1,2]

**Constraints:** 2 ≤ nums.length ≤ 10^4, exactly one solution, same element not used twice.

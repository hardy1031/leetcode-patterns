# 334. Increasing Triplet Subsequence

Return `true` if there exist indices i < j < k such that nums[i] < nums[j] < nums[k].

**Example 1:** nums = [1,2,3,4,5] → true
**Example 2:** nums = [5,4,3,2,1] → false
**Example 3:** nums = [2,1,5,0,4,6] → true

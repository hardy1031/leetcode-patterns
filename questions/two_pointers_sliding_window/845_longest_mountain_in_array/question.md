# 845. Longest Mountain in Array

Given an integer array `arr`, return the length of the longest subarray that is a mountain. Return 0 if none exists.

A mountain: arr[i-k] < ... < arr[i] > ... > arr[i+j], length ≥ 3.

**Example 1:** arr = [2,1,4,7,3,2,5] → 5
**Example 2:** arr = [2,2,2] → 0

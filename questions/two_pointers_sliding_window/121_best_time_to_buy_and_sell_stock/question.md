# 121. Best Time to Buy and Sell Stock

Given an array `prices` where `prices[i]` is the stock price on day `i`, return the maximum profit from one buy-sell transaction. Return 0 if no profit is possible.

**Example 1:** prices = [7,1,5,3,6,4] → 5
**Example 2:** prices = [7,6,4,3,1] → 0

# Analysis: 1. Two Sum

## Layer 3 — Problem-solving Pattern
**Pattern:** Hash Map usage (complement lookup)
One-pass: store each number's index in a hash map, check if `target - nums[i]` already exists.

## Layer 2 — Algorithms & Techniques
- Hash table lookup O(1)
- Brute force: nested loop O(n²)

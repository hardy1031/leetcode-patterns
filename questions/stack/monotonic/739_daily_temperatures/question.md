# 739. Daily Temperatures

You are given an array of integers `temperatures` where `temperatures[i]` represents the daily temperature on the ith day.

Return an array `result` where `result[i]` is the number of days after the ith day before a warmer temperature appears. If no warmer day exists in the future, set `result[i] = 0`.

## Examples

**Example 1:**
```
Input: temperatures = [30,38,30,36,35,40,28]
Output: [1,4,1,2,1,0,0]
```

**Example 2:**
```
Input: temperatures = [22,21,20]
Output: [0,0,0]
```

## Constraints
- `1 <= temperatures.length <= 10^5`
- `1 <= temperatures[i] <= 100`
